添加时间轴

reco_luan 2019-04-09

在 1.1.0 版本之后，对时间轴进行了重构，优化配置，用户无需额外添加相关文件，其他配置不变。

# 添加导航按钮

// .vuepress/config.js

module.exports = {
  theme: 'reco',
  themeConfig: {
    nav: [
      { text: 'TimeLine', link: '/timeline/', icon: 'reco-date' }
    ]
  }
}

# 添加所需的文件 1.1.0+ 无需要配置此项

/docs/timeLine/README.md

---
isTimeLine: true
sidebar: false
isComment: false
---
